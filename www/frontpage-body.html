VisiData

v0.99

Rapidly explore columnar data in the terminal

VisiData is free and can be installed in seconds on any Linux or MacOS system.

MacOS (brew)

Debian/Ubuntu (apt)

Other (pip3)

Efficient

Immediately start browsing large datasets while they load in the background. Every command is only one or two keystrokes.

Universal

Open gigabyte .csv files, remote Postgres databases, JSON endpoints, HDF5 files, and anything that can be viewed as tabular data. (See the list of supported sources.)

Extensible

Customize [options](/howto/options), add custom [commands](/howto/dev/commands), and add support for new [data sources](/howto/dev/loaders), with a minimum of code.
